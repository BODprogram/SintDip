«Приложение «Музыкальный синтезатор» с системой простого взаимодействия».

Приложение — синтезатор для составления мелодий по нотам с использованием различных наборов звуков и инструментов.
Это приложение предоставляет возможность создавать и настраивать наборы звуков различных инструментов и партитуры нот для проигрывания различных мелодий на различных инструментах простыми действиями.

Как пользоваться:
Редактор мелодий: чтобы собрать мелодию, необходимо указать название карты нот, не указывая тип файла. Посмотреть готовые карты нот можно в папке Maps. Также необходимо указать название набора звуков, так-же, не указывая тип файла. Проверить готовые наборы звуков можно в папке Soundpacks. Название итоговой мелодии тоже нужно указывать без указания типа файла. Мелодия появится в папке Export.
Редактор карт нот: Карта нот заполняется построчно, по 3 значения в строку. Первое это название ноты, такое же, как и в наборах звуков. Второе – время включения ноты, третье громкость. Ноты лучше располагать по порядку от начала мелодии до конца. В конце карты нот всегда идёт ключевое слово “end”, без дополнительных сопроводительных символов.  
Редактор наборов звуков: Для корректной работы указательный текстовый файл звукового набора должен лежать в папке Soundpacks, там-же должен лежать и одноимённая папка со звуками. Текстовый файл заполняется построчно, парами значений “название ноты” + “название файла ноты”. Название файла ноты указывается вместе с типом файла.
